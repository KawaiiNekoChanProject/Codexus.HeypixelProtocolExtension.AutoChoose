# Codexus.HeypixelProtocolExtension.AutoChoose
Heypixel 协议扩展

[Codexus.HeypixelProtocolExtension](https://github.com/KawaiiNekoChanProject/Codexus.HeypixelProtocolExtension) 的二级扩展

在部分状态下自动替用户做出选择

# 依赖

需要 [Codexus.HeypixelProtocolExtension](https://github.com/KawaiiNekoChanProject/Codexus.HeypixelProtocolExtension) 插件 (v0.0.3 及以上版本)

# 安装

1.从可靠来源获取插件

2.放入 `plugins` 目录
